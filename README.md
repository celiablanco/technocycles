# Technosphere Collapse and Technosignature Duty Cycles

Manuscript exploring how collapse-recovery dynamics in future Earth civilizations affect the detectability of technosignatures, with implications for the Drake Equation and the Great Silence.

## Layout
- `tex/` — LaTeX sources (`main.tex`, `preamble.tex`, `macros.tex`, and section files in `sections/`).
- `bibliography/` — BibTeX references (`references.bib`).
- `figures/` — PNG figures generated by the notebooks/scripts.
- `notebooks/` — Jupyter notebook (`figures.ipynb`) for running Monte Carlo simulations and generating main figures; `s8_sweep.py` for the S8 oscillatory parameter sweep figure.

## Build the manuscript
```bash
cd tex
pdflatex main && bibtex main && pdflatex main && pdflatex main
```

## Regenerate figures
- **Main figures** (trajectories + summary panel): open `notebooks/figures.ipynb` and run all cells.
- **S8 parameter sweep**: run `python notebooks/s8_sweep.py` from the project root.

Output goes to `figures/`.

## Scenarios
The simulations are based on ten 1000-year future scenarios from [Haqq-Misra et al. (2025)](https://doi.org/10.1016/j.techfore.2025.124194). Each scenario is parameterized with growth rate, resource stock, depletion rate, collapse depth, recovery delay, recovery fraction, and existential hazard rate. See Table 4 in the manuscript for full parameter values.
